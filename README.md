# 粤语拼音学习系统

基于《通用规范汉字表》的粤语拼音学习系统，提供8105个汉字的标准粤拼发音和学习功能。

## 🌐 在线使用

**在线地址**: https://wongtttads.github.io/Learning-Jyutping/

## ✨ 功能特点

- **章节学习**: 按照频率分成10个章节，从高频到低频逐步学习
- **标准发音**: 7990个汉字的预录制粤语发音音频
- **简洁界面**: 单页面应用，无刷新体验
- **响应式设计**: 完美适配手机、平板、电脑

## 📊 数据统计

- 总汉字数: 8105
- 学习章节: 10章
- 发音音频: 7990个
- 数据来源: 《通用规范汉字表》
- 拼音方案: 香港语言学学会官方粤拼 (Jyutping)

## � 快速开始

1. 打开在线地址: https://wongtttads.github.io/Learning-Jyutping/
2. 选择一个章节开始学习
3. 点击汉字卡片上的"朗读"按钮听发音
4. 建议每日学习80-100字

## � 项目结构

```
Learning-Jyutping/
├── index.html              # 单页面应用入口
```

## 🎯 核心模块

### 数据管理器 (DataManager)
- 加载章节数据
- 加载音频索引
- 查找音频路径
- 提供统计信息

### 发音系统 (PronunciationSystem)
- 查找音频文件
- 播放音频
- 停止播放
- 状态管理

### UI渲染器 (UIRenderer)
- 渲染章节列表
- 渲染汉字卡片
- 处理点击事件
- 显示/隐藏视图

### 主应用 (App)
- 初始化所有模块
- 协调各模块交互
- 错误处理

## 🔧 技术栈

- **前端**: Vanilla JavaScript (ES6+)
- **样式**: CSS3 + Flexbox/Grid
- **数据**: JSON文件
- **部署**: GitHub Pages
- **音频**: MP3格式

## 📝 开发说明

本项目采用模块化设计，各模块职责清晰：

1. **单一职责**: 每个模块只负责一个功能
2. **零依赖**: 不使用外部框架，减少加载时间
3. **渐进增强**: 核心功能优先，可选功能后加
4. **性能优先**: 懒加载音频，按需加载数据

## 📄 许可证

本项目数据来源于《通用规范汉字表》，仅供学习使用。
