# 粤语拼音学习系统

一个基于《通用规范汉字表》的粤语拼音学习系统，提供完整的汉字拼音数据、发音功能、学习测验和进度跟踪。

## 🌐 在线使用

### 访问在线版本

本项目已部署到GitHub Pages，可以直接在浏览器中访问使用：

**🔗 在线地址**: https://wongtttads.github.io/Learning-Jyutping/

### 在线功能

在线版本提供以下完整功能：

#### 📚 学习功能
- **分章节学习**: 按照频率分成的10个章节，从高频到低频逐步学习汉字和粤语拼音
- **全屏学习**: 专注模式，减少干扰，专注于汉字和拼音的学习
- **汉字卡片**: 每个汉字卡片显示汉字、拼音、声调、频率等信息
- **例词参考**: 提供常用例词帮助理解汉字用法
- **点击发音**: 点击发音按钮即可听到标准粤语发音

#### 🎯 测验系统
- **汉字→拼音测验**: 根据汉字选择正确的粤语拼音
- **拼音→汉字测验**: 根据拼音选择正确的汉字
- **混合模式测验**: 随机混合两种类型
- **自定义题目数量**: 可以选择10、20、50道题目
- **实时反馈**: 即时显示答题结果
- **错题回顾**: 测验结束后查看错题和正确答案

#### 📊 进度跟踪
- **学习统计**: 查看已学习汉字数量、测验次数、平均分数
- **连续学习天数**: 记录连续学习天数，激励持续学习
- **章节进度**: 查看各章节的学习进度
- **测验历史**: 查看历史测验记录
- **智能复习提醒**: 提示需要复习的汉字

#### 🔊 发音功能
- **预录制音频**: 7990个汉字的粤语发音音频，发音质量一致
- **快速响应**: 预录制音频播放速度快，无需等待
- **跨平台兼容**: 支持所有现代浏览器
- **离线可用**: 音频文件可以缓存，无需网络连接

### 使用方法

1. **开始学习**: 打开在线地址，选择"分章节学习"或"全屏学习"
2. **选择章节**: 从第1章到第10章中选择适合的章节
3. **学习汉字**: 浏览汉字卡片，点击发音按钮听发音
4. **进行测验**: 点击"测验系统"检验学习效果
5. **查看进度**: 点击"学习进度"查看学习统计和历史记录

### 技术特点

- **纯前端实现**: 无需后端服务器，所有功能在浏览器中运行
- **预录制音频**: 7990个汉字的粤语发音音频，确保发音质量
- **响应式设计**: 支持桌面、平板、手机等多种设备
- **数据持久化**: 使用LocalStorage保存学习进度
- **快速加载**: 优化的数据结构和缓存机制

## 快速开始

### 环境要求

- Python 3.7+
- macOS系统（发音功能需要）
- 现代浏览器（Chrome、Firefox、Safari等）

### 安装依赖

```bash
pip install -r requirements.txt
```

### 数据处理

1. 构建主数据库（下载汉字表并转换拼音）：

```bash
python scripts/build_master_db.py
```

2. 生成章节数据：

```bash
python scripts/generate_char_data.py
```

3. 生成例词（推荐使用改进版）：

```bash
python scripts/generate_example_words_v3.py
```

### 启动发音服务器

```bash
python server.py
```

服务器将在 `http://localhost:5001` 启动。

### 使用学习系统

1. 打开 `output/frequency_chapters.html` 开始学习
2. 打开 `output/quiz.html` 进行测验
3. 打开 `output/progress.html` 查看进度

## 使用说明

### 学习模式

### 进度管理

2. 查看学习统计

## 技术栈

- **后端**：Python + Flask
- **前端**：HTML5 + CSS3 + JavaScript
- **数据处理**：Pandas + ToJyutping
- **发音**：macOS say命令 + Sinji语音包
- **存储**：LocalStorage（前端进度）

## 注意事项

1. **发音功能**：需要macOS系统并安装粤语语音包（Sinji）
   - 检查语音包：`say -v '?' | grep Sinji`
   - 如果没有安装，请在系统设置中添加

2. **数据来源**：汉字表来自GitHub开源项目，可能存在个别字拼音不准确

3. **多音字处理**：当前只取第一个拼音候选，后续版本会改进

4. **浏览器兼容性**：建议使用现代浏览器以获得最佳体验
